# User Context

- Building a SaaS platform called **TrekOps / TrailDesk** for trek tour operators in India
- The system manages WhatsApp-based bookings for trek companies (customers interact via WhatsApp, operators manage via web admin)
- Multi-tenant platform — single deployment serves multiple trek companies
- Strong technical involvement — active codebase with regular commits across frontend and backend
- Project is in active deployment (`Deploy/V1` branch, production at `https://api.trekops.in`)
- Primary market: Indian trek companies (Maharashtra region focus — Pune, Mumbai area)
- Payment gateway: PayU (Indian payment processor)
- WhatsApp integration: Meta Business API (real WhatsApp, not third-party)
- AI integration: Groq (Llama 3.3 70B) for NLU on unrecognized WhatsApp messages
